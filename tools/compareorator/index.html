IBM Cloud for AWS professionals

This article will hopefully help anyone coming from the AWS ecosystem translate their workloads into IBM Cloud.

Basic Concepts

IBM Cloud has three main silos. Infrastructure , Applications , and Services.

Infrastructure is built on SoftLayer and is available in over 30 SoftLayer Datacenters across the world. All of which are interconnect with our unmetered private network.

Applications can be either Cloud Foundry, Docker, or Open Whisk

Services contain a huge catalog of pay as you go addons to your application, such as Watson, Databases and Analytics to name a few.

Network

On IBM Cloud Infrastructure (SoftLayer) you are only charged for bandwidth that leaves your account. Incoming traffic, and everything on the private network is unmetered, allowing you to take full advantage of our worldwide private network for shipping data between locations. For more information on bandwidth costs see our pricing page.

See Deconstructing SoftLayer’s Three Tiered Network for some more information on how it all works.

Bare Metal

Standard-configuration servers available within 20-30 minutes, and fully customizable servers available within 4 hours. Power8 servers are available as well, built on IBM OpenPOWER architecture. GPU cards and SAP certified servers are also available

Watson

With Watson on IBM Cloud, you have access to the widest range of cognitive technologies available today to quickly and securely build smart applications. From analyzing images and video to understanding sentiment, keywords and entities from text, our Watson services enable cognitive within your applications.

API

Everything in IBM Cloud can be managed with an API, for infrastructure this is done through the SoftLayer API and IBM Cloud Apps and Services can be managed through the IBM Cloud CLI which is an implementation of the Cloud Foundry API

Pricing

Most IBM Cloud services have a very aggressive free trail / tier to get you started, and transparent pricing plans for when you need to scale up. More information on IBM Cloud Pricing and SoftLayer Pricing Calculator
