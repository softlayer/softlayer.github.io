Create an virtual guest
Creates a new virtual guest with some basic options

Operation: POST

Method: SoftLayer_Virtual_Guest::createObject()

URL: SoftLayer_Virtual_Guest

Example CURL: curl -u username:api_key --data @createVirtualServer.json "https://api.softlayer.com/rest/v3/SoftLayer_Virtual_Guest"

Input JSON:

{ 
    "parameters":[ 
    { 
        "datacenter" : {
            "name" : "sjc01"

        "hostname": "new", 
        "domain": "Test.com", 
        "startCpus": 1, 
        "maxMemory": 1024,

        "operatingSystemReferenceCode": "DEBIAN_LATEST" ,

    ] 
}
